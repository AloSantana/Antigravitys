  🚀 QUICK START GUIDE
═══════════════════════════════════════════════════════════════════════════════

1. Install Dependencies:
   $ cd backend
   $ pip install -r requirements.txt

2. Start Backend:
   $ uvicorn main:app --reload --host 0.0.0.0 --port 8000

3. Open Frontend:
   Navigate to: http://localhost:8000

4. Access Settings:
   Click "⚙️ Settings" tab in navigation

5. Configure:
   • Add API keys (Settings > API Keys Management)
   • Test connections (Settings > AI Model Configuration)
   • Enable MCP servers (Settings > MCP Server Manager)
   • Configure server (Settings > Server Configuration)
